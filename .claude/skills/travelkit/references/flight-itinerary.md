# flight-itinerary ref

## flight_download_itinerary

下载已出票订单的行程单（电子行程单 / PDF / ZIP）。

### 何时使用

用户主动请求行程单或类行程单票据文件，且订单状态支持下载时。

**不主动**在订单查询或其他操作结束后推送行程单下载。

### 使用方式

调用 `flight_download_itinerary` 时，传入订单号。

请求参数中的 `type` 会影响常见返回格式：

- `type=more`：生成合并文件，通常返回单个 PDF
- `type=once`：可能生成多个文件，多个文件时可能返回 ZIP 压缩包

下载完成后，说明返回文件的类型（工具返回该信息时）：

- 是 PDF 文件
- 是 ZIP 压缩包
- 还是其他格式

最终格式以工具返回的文件类型、响应 `Content-Type` 或文件名后缀为准。工具未返回文件类型信息时，如实说明未返回，不要猜测为 PDF。

### 错误处理

下载失败或订单状态不支持下载时，简短说明原因，建议用户核对订单状态或稍后重试。
